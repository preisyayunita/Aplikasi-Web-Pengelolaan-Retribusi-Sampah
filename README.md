# Aplikasi Pengelolaan Retribusi Sampah berbasis Web
Aplikasi Pengelolaan Retribusi ini di buat menggunakan CodeIgniter 4. Aplikasi Ini dirancang untuk membantu pegawai sebagai Admin dalam memudahkan pengelolaan Retribusi Sampah, kemudian dapat membantu masyarakat daftar melalui online jadi tidak perlu pergi ke Kantor.
## Fitur
1. Registrasi
2. Daftar Berlangganan
3. Login
4. Dashboard sebagai Monitoring
5. Kelola Kategori oleh Admin
6. Validasi Form Pelanggan Baru oleh Admin
7. Validasi Bukti Pembayaran oleh Admin
8. Monitoring dan Printing Laporan Retribusi Sampah via .pdf dan .xlsx oleh Admin
9. Kelola SMS oleh Admin
10. Lihat Riwayat Transaksi/Pembayaran oleh Pelanggan
11. Kirim Bukti Transaksi/Pembayaran oleh Koordinator Lapangan
